JavaScript 编码风格指南

 这几天一直在读这本《编写可维护的JavaScript》 ,觉得编码风格对于一个长期维护的软件而言是非常的重要的。下面我就列举一下书中建议的一些规范(下面的写法均为推荐写法)。

缩进

每一行由四个空格组成,避免使用Tab进行缩进。

行的长度

每行不应超过80个字符。如果一行多于80个字符，应当在一个运算符后换行。下一行增加两级缩进（8个字符）
